# Quick Projects & Configuration Files Navigator

<!-- ![p](https://github.com/Pheon-Dev/flamingo/blob/main/flamin.gif) -->

## Configuration file

```bash
~/.config/p/config.yaml
```

### Example

```yaml
---
filtering: true
projects:
  - title: nvim
    description: ~/.config/nvim
    path: .config/nvim
  - title: dwm
    description: ~/.config/arco-dwm
    path: .config/arco-dwm
status-bar: true
title: Flamingo
```
